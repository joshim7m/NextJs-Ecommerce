# Roadmap & Milestones

## Phase 1 — Foundation
- Prisma schema and seed data
- Storefront skeleton: home, category, product pages
- Cart persistence (session)

## Phase 2 — Admin
- Admin authentication
- Category and product CRUD

## Phase 3 — Polish & Deploy
- Testing, SEO, accessibility
- Deploy to staging and production

## Acceptance Criteria per Milestone
- Data & API: schema deployed and seeded
- Storefront MVP: pages render and cart works
- Admin MVP: create/edit product reflects in storefront
